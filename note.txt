到 GitHub 專案頁面，點擊綠色的 Code 按鈕。

複製裡面的網址（HTTPS 或 SSH 都可以）。

在終端機 git clone https://github.com/帳號/專案名稱.git

提交變更
git add .
git commit -m "新增功能或修改說明"

上傳回 GitHub / 保持同步
git push


# [前後端同步執行]
npm i -D concurrently
直接在/frontend 目錄 下 npm run dev

| 項目               | 檢查方式                                                          | 正常結果                                     |

| 若出現錯誤         | 看 concurrently log 標籤 `[BACK]` 或 `[FRONT]`                     | 可獨立 debug                                |
